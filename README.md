# VRCX-PTBR

This repository contains the Brazilian Portuguese (pt-BR) translation of **VRCX**, managed through Pontoon.

## About

The purpose of this repository is to facilitate the translation of VRCX into Brazilian Portuguese using Pontoon as the primary platform. Due to Pontoon's structure, translations are handled via GitHub directories, which is why this repository exists.

This translation is not restricted to a single contributor. Anyone is free to contribute here or directly to the official repository.

This project is mainly intended to support my own translation workflow, but contributions are welcome from anyone interested in helping improve the localization.

## Translation Platform

You can access and contribute to the translation through Pontoon here:  
👉 https://tl.inexo.cloud/pt/vrcx-ptbr/

## Official Project

All translations are ultimately intended for integration into the official VRCX project:  
👉 https://github.com/vrcx-team/VRCX

## Contributing

Contributions are optional but appreciated. Feel free to participate if you would like to help improve the pt-BR translation.
